# Documentation-VDS
Description des ressources documentaires, matérielles et logicielles utilisées

## Description
L'amicale de Val de Somme est une association sportive de course à pied qui dispose actuellement d'un
site vitrine sur lequel elle souhaite intégrer des modules supplémentaires permettant de rendre le site
plus dynamique. 
Vous êtes chargé de participer à l’évolution de ce site

## Documentation mise à la disposition des étudiants

- Cahier des charges
- Etude de l'existant
- Mise en place de l'application

## Documentation des modules à réaliser

- Les documents
- Les événements
- Les informations
- Les inscriptions en ligne
- Les partenaires

Le sous répertoire document contient les documents pdf à intégrer
Le sous répertoire partenaire contient les logos des partenaires à intégrer
